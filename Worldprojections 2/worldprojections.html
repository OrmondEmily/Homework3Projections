In this project I learned how to display images in different projections

How I displayed the map in different projections using QGIS

I first imported the map shapefile from the data provided on Sakai. I then changed the projection by clicking on the bottom right coner where the ESPG was displayed. I then found each projection by typing in its ESPG code in the search bar. I used the plug in "Inticatrix Mapper" to show how the different projectioins distored the map
